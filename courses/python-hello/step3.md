# 运算与表达式

Python 不仅可以存储数据，还能进行各种运算。

## 算术运算

```python
# 加减乘除
a = 10
b = 3
print(a + b)   # 加法: 13
print(a - b)   # 减法: 7
print(a * b)   # 乘法: 30
print(a / b)   # 除法: 3.333...
print(a // b)  # 整除: 3
print(a % b)   # 取余: 1
print(a ** b)  # 幂运算: 1000
```

## 字符串运算

```python
first = "Hello"
second = "World"
print(first + " " + second)  # 字符串拼接: Hello World
print("-" * 20)             # 字符串重复: --------------------
```
