# 运算与表达式

Python 不仅可以存储数据，还能进行各种运算。

## 算术运算

在右侧编辑器中输入：

```python
a = 10
b = 3

print("a + b =", a + b)   # 加法: 13
print("a - b =", a - b)   # 减法: 7
print("a * b =", a * b)   # 乘法: 30
print("a / b =", a / b)   # 除法: 3.333...
print("a // b =", a // b)  # 整除: 3
print("a % b =", a % b)   # 取余: 1
print("a ** b =", a ** b)  # 幂运算: 1000
```

## 运算符优先级

| 优先级 | 运算符 | 说明 |
|--------|--------|------|
| 高 | `**` | 幂运算 |
| 中 | `*`, `/`, `//`, `%` | 乘除整除取余 |
| 低 | `+`, `-` | 加减 |

## 字符串运算

```python
first = "Hello"
second = "World"

# 字符串拼接
print(first + " " + second)  # Hello World

# 字符串重复
print("-" * 20)             # --------------------

# 格式化输出
name = "小明"
age = 18
print(f"姓名: {name}, 年龄: {age}")
```

## 复合赋值运算符

```python
x = 10
x += 5      # x = x + 5
print(x)    # 15

y = 20
y -= 3      # y = y - 3
print(y)    # 17
```
