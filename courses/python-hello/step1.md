# Hello World

每个程序员的第一步就是输出 "Hello World"。在 Python 中，我们使用 `print()` 函数来输出内容。

## 你的第一个程序

在右侧的代码编辑器中，输入以下代码：

```python
print("Hello, World!")
```

然后点击上方的「执行」按钮运行代码。

## 语法说明

| 元素 | 说明 |
|------|------|
| `print()` | Python 的内置输出函数 |
| `"..."` 或 `'...'` | 字符串字面量 |
| `()` | 函数调用括号 |

## 试一试

修改代码，将 "Hello, World!" 改成其他内容，比如：
- `print("你好，世界！")`
- `print("Python 真棒！")`
- `print(123 + 456)`

点击「执行」看看结果！
