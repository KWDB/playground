# 变量与数据类型

变量是存储数据的容器。在 Python 中，创建变量非常简单。

## 创建变量

```python
name = "小明"
age = 18
height = 1.75
is_student = True
```

## 数据类型

| 类型 | 示例 | 说明 |
|------|------|------|
| str | "Hello" | 字符串（文本） |
| int | 18 | 整数 |
| float | 1.75 | 浮点数 |
| bool | True/False | 布尔值 |

## 打印变量

```python
print(name)
print(age)
print("姓名:", name, "年龄:", age)
```
