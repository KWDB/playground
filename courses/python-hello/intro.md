# Python 入门课程

欢迎来到 Python 编程入门课程！本课程将带您了解 Python 的基础知识。

## 课程目标

- 掌握 Python 基本语法
- 理解变量和数据类型
- 学会使用 print() 输出
- 了解基本运算

## 如何使用

本课程采用交互式学习方式：

1. **阅读内容** - 左侧查看课程讲解
2. **编写代码** - 在右侧代码编辑器中输入代码
3. **执行代码** - 点击「执行」按钮运行
4. **查看结果** - 在下方查看输出

## 什么是 Python？

Python 是一种简单易学、功能强大的编程语言。它拥有简洁的语法和丰富的标准库，非常适合初学者学习编程。

特点：
- 语法简洁易读
- 应用领域广泛（Web、数据分析、AI 等）
- 强大的标准库
