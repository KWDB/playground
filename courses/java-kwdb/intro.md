## 欢迎来到 KWDB 课程

欢迎学习使用 Java 连接 KWDB 数据库！本课程将带你了解如何使用 JDBC 连接 KWDB，并执行常见 SQL 操作。

## 课程目标

- 掌握使用 Java JDBC 连接 KWDB
- 学习创建数据库和表
- 掌握数据的插入与查询

## 预备知识

- 了解 Java 基础语法
- 了解 JDBC 基础概念
- 了解 SQL 基础知识

## 课程环境

- KWDB 服务运行在 `127.0.0.1:26257`（容器内）
- 数据库用户：`root`
- 密码：空（`--insecure` 模式）
- JDBC URL 示例：`jdbc:kaiwudb://127.0.0.1:26257/defaultdb`
